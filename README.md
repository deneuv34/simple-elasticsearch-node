# Simple NodeJS API Server

## Installation

### Requirements
- Elasticsearch >= v2.x
- MySql
- Node >= 6
- Apache2 / Nginx (recommended with PHPMyAdmin)

### Setting Up
- Clone this repo.
- Install all dependency using `npm install` or `yarn`.
- Run app using `node index.js`.
